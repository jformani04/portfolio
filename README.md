## Introduction

## Live Website

[https://portfolio-wine-three-s554mye2sz.vercel.app/](https://portfolio-wine-three-s554mye2sz.vercel.app/)

## Features

- Modern responsive design
- Dedicated resume page at `/resume`
- Downloadable resume from the resume page
- Clean UI built with Tailwind CSS
- Component-based architecture using Next.js
- Optimized for performance and deployment on Vercel

## Tech Stack

- Next.js
- Tailwind CSS
- TypeScript
- Vercel

## Project Structure

- `app/` - App Router pages, layouts, global styles, and route-specific pages such as `/resume`
- `components/` - Reusable UI components for sections like About, Experience, Projects, Skills, and Contact
- `public/` - Static assets including the resume download file and other public resources

## Getting Started

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Then open `http://localhost:3000` in your browser.

## Deployment

This portfolio is deployed on Vercel, which provides fast hosting, production builds, and seamless deployment for Next.js applications. Changes pushed to the configured repository can be deployed efficiently through the Vercel platform.
